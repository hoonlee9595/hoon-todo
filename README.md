# 할일 관리 앱 (Todo App)

Firebase Realtime Database를 사용한 할일 관리 웹 애플리케이션입니다.

## 주요 기능

- ✅ 할일 추가
- ✏️ 할일 수정 (인라인 편집)
- 🗑️ 할일 삭제
- ☑️ 완료 상태 토글
- 🔍 필터링 (전체/진행중/완료)
- 🧹 완료된 항목 일괄 삭제
- 🔄 실시간 동기화 (Firebase Realtime Database)

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- Firebase Realtime Database

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 할일을 입력하고 추가 버튼 클릭
3. 수정 버튼을 클릭하여 할일 내용 수정
4. 체크박스를 클릭하여 완료 상태 변경
5. 삭제 버튼을 클릭하여 할일 삭제

## Firebase 설정

Firebase Realtime Database를 사용하기 위해 `script.js` 파일의 Firebase 설정을 확인하세요.
